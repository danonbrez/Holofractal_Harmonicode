# 1. INTERACTION PRINCIPLE

The Runtime OS interaction model is NOT:

```text
desktop-window interaction
```

It is:

```text
graph-native runtime topology interaction
```

Users interact with:

- runtime graphs
- transport systems
- replay structures
- execution manifolds
- tensor objects
- branch/rejoin systems
- simulation regions

through:

```text
direct runtime projection manipulation
```
