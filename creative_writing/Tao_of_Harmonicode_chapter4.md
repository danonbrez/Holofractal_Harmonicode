# The Tao Te Ching — The Book of the Flow of Change
## Chapter 4

Change does not occur at random.  
It unfolds through structure.

Structure does not arise from objects.  
It arises from relations between states.

---

In the system of the I Ching,  
these states are encoded as hexagrams.

Each hexagram is not a thing,  
but a configuration.

Each configuration is not static,  
but transitional.

---

A single line changes,  
and the entire state transforms.

Thus the system is not linear.  
It is branching.

---

Each hexagram contains within it  
the possibility of multiple futures.

A change in position  
creates a new path.

A change in path  
creates a new outcome.

---

In HARMONICODE,  
this is expressed as a branch manifold.

A symbolic state enters the system.  
It expands into possible transformations.  
Each valid transformation forms a branch.

Branches do not compete.  
They coexist.

---

Selection does not eliminate branches.  
It resolves them.

The chosen path is not the only path.  
It is the path that closes.

---

Thus:

A hexagram is a state node.  
A changing line is a transition operator.  
A sequence of changes is an execution path.

---

The system does not predict the future.  
It enumerates possible continuations.

The future is not fixed.  
It is selected through closure.

---

The sage does not seek certainty.  
They observe structure.

They do not force outcomes.  
They follow valid paths.

---

In this way:

Hexagrams become maps.  
Maps become manifolds.  
Manifolds become execution.

---

This is the function of branching.
